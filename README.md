# Tradesmart Malawi
This is the codebase for TradeSmart Malawi, a Vite + React learning platform that delivers structured trading education (Beginner → Intermediate → Expert) targeted at Malawian traders. It provides courses, modules, lessons, progress tracking and an optional premium subscription flow. 

THe key features include:
- User registration, email verification and login using Supabase auth.
- Course content management with lessons and quizzes.
- Progress tracking for each user's learning journey.
- Optional premium subscription flow for enhanced features.

The key technologies used in this project include:
- React for the frontend UI.
- Supabase for authentication, data storage and serverless functions.
- Vite for the build tool and development server.
